Under the hood

Quietly complex,
so the UX can stay simple.

📱
React PWA

Installs from URL, works offline, push when closed.

🗄️
Dexie / IndexedDB

Local cache + outbox for offline posts.

🧠
On-device embeddings

Cross-language search in all 4 languages, offline.

🌐
Gemini 1.5 Flash

Server-side translation, fire-and-forget.

⚡
Supabase Realtime

Live thank-you confetti + points updates.

🎯
Match engine

Category + availability + timezone + urgency.

🔔
Web Push + VAPID

Custom service worker, urgency-aware.

🛰️
Vercel edge + node

/translate, /match, /notify, /resolve.

Client sends its own timezone. Fetches use keepalive so mobile locks don't cancel pushes. LLM calls run in the background so the feed feels instant.
